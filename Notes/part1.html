Inversion sampling

Inversion sampling is a method for generating random samples from a target distribution \(p(x)\) by using its cumulative distribution function (CDF) \(F(x)\). The algorithm works as follows: 1. Compute the CDF \(F(x) = \int_{-\infty}^{x} p(t) dt\). 2. Generate a uniform random number \(u\) from the interval [0, 1]. 3. Find the value \(x\) such that \(F(x) = u\). This can be done by computing the inverse CDF, denoted as \(F^{-1}(u)\), which gives us the sample from the target distribution.

Rejection sampling

Rejection sampling is a method for generating random samples from a target distribution \(p(x)\) by using a proposal distribution \(q(x)\) that is easy to sample from. The algorithm works as follows:

  1. Choose a proposal distribution \(q(x)\) and a constant \(M\) such that \(M q(x) \geq p(x)\) for all \(x\).
  2. Repeat the following steps until you have the desired number of samples:
    1. Sample \(x'\) from the proposal distribution \(q(x)\).
    2. Generate a uniform random number \(u\) from the interval [0, 1].
    3. If \(u < \frac{p(x')}{M q(x')}\), accept \(x'\) as a sample from \(p(x)\); otherwise, reject it and repeat the process.

Theoretical Acceptance rate \[ \mathbb E\Big[U<\frac{p(X)}{M q(X)}\Big | X\Big] = \int \frac{p(x)}{M q(x)} q(x) dx = \frac{1}{M}\overbrace{\int p(x) dx}^{=1} = \frac 1M \]

The efficiency of rejection sampling depends on how well the proposal distribution \(q(x)\) approximates the target distribution \(p(x)\). If \(M\) is large, the acceptance rate will be low, leading to many rejected samples. Therefore, it is crucial to choose a proposal distribution that closely matches the shape of the target distribution to improve efficiency.

Example

Suppose you want to sample from \[ p(x)= \begin{cases} 2e^{-x}(1-e^{-x}) & \text{if } x \geq 0 \\ 0 & \text{otherwise} \end{cases} \]

having \(q(x) = e^{-x}\) for \(x \geq 0\) as proposal distribution.

Code 
def p_function(x):
    x = np.asarray(x)
    return np.where(x >= 0, 2 * np.exp(-x) * (1 - np.exp(-x)), 0.0)

def q_function(x):
    x = np.asarray(x)
    return np.where(x >= 0, np.exp(-x), 0.0)

x = np.linspace(0, 10, 1000)
yp = p_function(x)
yq = q_function(x)

We have to choose a constant \(M\) such that \(Mq(x) \geq p(x)\) for all \(x\): \[ M = \max_{x \geq 0} \frac{p(x)}{q(x)} = \max_{x \geq 0} \frac{2e^{-x}(1-e^{-x})}{e^{-x}} = \max_{x \geq 0} 2(1-e^{-x}) = 2 \]

We can now try to run point 2b - 2c until we reach 1000 accepted samples:

Code 
n_samples = 1000
accepted_x, accepted_y = [], []
rejected_x, rejected_y = [], []
it = 0                    # iterations counter
while len(accepted_y) < n_samples:
    x_prime = np.random.exponential(scale=1.0)
    u = np.random.uniform(0, 1)
    
    # This is the actual height of the point in the proposal distribution
    y_val = u * Mq_function(x_prime)
    
    # Check against the target p(x)
    if u < p_function(x_prime) / Mq_function(x_prime):
        # Accepted
        accepted_x.append(x_prime)
        accepted_y.append(y_val)
    else:
        # Rejected
        rejected_x.append(x_prime)
        rejected_y.append(y_val)
        
    it += 1

Interactive rejection sampling

viewof lambda = Inputs.range([0.1, 3], {step: 0.1, label: "λ (Proposal)", value: 1})

Code 
// 1. The Simulation Logic
// This runs whenever the 'run' button is clicked
results = {
  run; // Reference 'run' to trigger recalculation
  const n = 500;
  const accepted = [];
  const rejected = [];
  let attempts = 0;

  while (accepted.length < n && attempts < 5000) {
    attempts++;
    // Sample x from exponential proposal: x = -ln(1-u)/lambda
    const x_prime = -Math.log(1 - Math.random()) / lambda;
    const u = Math.random();
    
    const p_val = x_prime >= 0 ? 2 * Math.exp(-x_prime) * (1 - Math.exp(-x_prime)) : 0;
    const mq_val = M * (lambda * Math.exp(-lambda * x_prime));
    
    const point = {x: x_prime, y: u * mq_val};
    
    if (u < p_val / mq_val) {
      accepted.push({...point, status: "Accepted"});
    } else {
      rejected.push({...point, status: "Rejected"});
    }
  }
  return {accepted, rejected, rate: accepted.length / attempts};
}
